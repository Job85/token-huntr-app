# token_huntr
Become the Jack Sparrow of Geocaching! Track down and discover the exact location of a geocache to earn valuable tokens. Some tokens are worth more depending on their level of difficulty in finding.

## Technology Used
- ** Postresql
- ** Express
- ** React
- ** Node
